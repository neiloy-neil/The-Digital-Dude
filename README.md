# The Digital Dude - Official Website

This is the official company website for The Digital Dude, a bespoke software development and AI solutions agency.

## Overview

This project serves as the primary digital presence for The Digital Dude. It showcases our services, highlights our successful projects through detailed case studies, and provides a point of contact for potential clients. The website is built with a modern tech stack to reflect our commitment to cutting-edge technology.

## Tech Stack

-   **Framework**: React with TypeScript
-   **Styling**: Tailwind CSS
-   **Animations**: Framer Motion
-   **UI Components**: Swiper.js for carousels, Lucide for icons.
-   **Build Tool**: Vite

## Project Structure

-   `src/components`: Reusable UI components and page sections.
-   `src/pages`: Top-level page components representing different views of the SPA.
-   `src/data`: Centralized static content for services, case studies, etc.
-   `src/styles`: Global styles and Tailwind CSS configuration.
